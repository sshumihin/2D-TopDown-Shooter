# 2D-TopDown-Shooter
This game made as a challenge.

**How to play:**
- Move - WASD or arrows
- Shoot - Left Mouse Button

- Find keys to open grey doors.
- Shoot brown doors to loot aids.
- Avoid robots, shoot them, kill them or you will die.
- Find the EXIT.
<br>
